//多人戰鬥環境下進行測試

麻煩留意BOSS戰鬥當中是否有下列情形發生 : 
	1.鬼隱 : 戰鬥中途消失、鑽地消失或無故消失
	2.中途退出 : 中途退出戰鬥 或 發呆看電視	( 留意發生時 BOSS身上光環是 紅色 還是 黃色 )
	3.牆隱 : 卡進牆裡 ( 尤其是鑽地竄出 或 衝刺 後 )

測試進行方式 :
	A.前置動作 : 依序輸入下列指令
		1.	/? world 0 318 1	( 移動至冒險區 )
		2.	/? setvalue lv 1		( 設定等級為1 )
		3.	/? setvalue VIT 999	( 設定血量 )
		4.	/? reseteq 1		( 設定裝備 )
		5.	/? gotoplayer Aris

			
	B.戰鬥進行
		Stage 1 : 不將BOSS打死
			1.遠程攻擊者不斷移動攻擊
			2.將BOSS誘至牆邊戰鬥
			3.持續將BOSS誘至不同戰鬥位置
